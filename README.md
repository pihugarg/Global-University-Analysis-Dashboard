# 📌 Project Overview

The **Global University Analysis Dashboard** is a comprehensive **Power BI Business Intelligence project** that analyzes universities across the world using global ranking datasets.

The dashboard enables users to explore **country-wise academic performance, university rankings, student demographics, ranking systems, and yearly trends** through interactive visualizations and dynamic filtering.

It has been designed to provide meaningful insights for students, researchers, educational institutions, and policy makers while demonstrating advanced Power BI capabilities.

---

# 🎯 Objectives

- Compare university performance across countries.
- Analyze global ranking systems.
- Explore student demographics and diversity.
- Track yearly changes in rankings and enrollment.
- Build an interactive business intelligence solution using Power BI.

# 📂 Dataset

The project utilizes six interconnected datasets.

| Dataset | Description |
|----------|-------------|
| Country | Country information |
| University | University details |
| University Year | Student demographics by year |
| University Ranking Year | Ranking scores across years |
| Ranking System | Ranking organizations |
